# 原型

`diviflow_interactive_1.html` 是 DiviFlow 的高保真交互原型，单文件 HTML，覆盖 17+ 屏：

| 分组 | 屏幕 |
| --- | --- |
| 入口 | 启动页、登录、注册 |
| Onboarding | 地区选择、当前资产、月支出目标 |
| 核心 | 主看板 Home、持仓列表、添加持仓、添加成功确认 |
| AI | AI 扫描持仓截图（上传 → 识别中 → 结果确认） |
| 股票 | 股票搜索 |
| 财务自由 | FI 追踪、分红日历 |
| 辅助 | 消息通知、成就打卡、发现工具、设置/我的、空状态引导 |

## 使用

在浏览器中直接打开即可。UI 视觉与设计令牌（颜色 / 字号 / 圆角）是前端实现时的主要参考。
